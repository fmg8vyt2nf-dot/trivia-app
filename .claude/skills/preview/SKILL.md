# Preview & Iterate Skill

When invoked:

1. Open the current app in Claude Preview (start dev server if needed)
2. Take a screenshot and describe what you see
3. Check for: layout issues, interactive elements working, visual polish
4. List 3 suggested improvements
5. Ask user which improvements to make
